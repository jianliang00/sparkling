# Reporting a Vulnerability

If you happen to discover a potential security issue in this project, please do **not** create a public GitHub issue.

We kindly ask you to notify TikTok Security through the [Sparkling-TikTok topic](https://discord.com/channels/1345754526174543964/1499022379052171376) in the Lynx Discord channel or via email at [security@tiktok.com].

We greatly appreciate community contributions to the security of this project.
